Twentytwelve WordPress Child Theme using LESS CSS

Simple Twentytwelve WordPress child theme using LESS.

Based on the demo code used for the WordPress NYC Meetup Presentation on 3/18/2014: Better WordPress Style: Using SASS and LESS CSS Preprocessors

Yes, it's a 90s style theme, complete with gradients and box shadows, in lovely shades of green. :)
